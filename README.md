# tball-soundboard-deploy

Deploy-only branch for the Walk-Up! T-ball soundboard. The repo root of
this branch is the static site Vercel serves (Production Branch =
`tball-soundboard-deploy`), sidestepping the Root Directory setting.

Do not edit here — the source of truth is `tball-soundboard/` on `main`.
This branch is regenerated from that folder when the app changes.
